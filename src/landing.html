HOW IT WORKS

To start using tinyCam Cloud do the following:
VIA WEB BROWSER
  1. Register on tinyCam Cloud service.
  2. Login to tinyCam Cloud and add your cameras to the cloud in Account tab.
  3. Make subscription.
ON ANDROID
  1. Register on tinyCam Cloud service.
  2. Install Android app called tinyCam Monitor FREE or PRO from Google Play or Amazon AppStore.
  3. Enable tinyCam Cloud in tinyCam Monitor app settings.
  4. Select tinyCam Cloud in tinyCam Monitor app drawer, enter login credentials, and add your IP cameras to the cloud in Account tab.
  5. Make subscription via web browser.
After you have added the camera, tinyCam Cloud should start 24/7 video/audio recording and motion detection.
